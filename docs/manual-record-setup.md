# 수동 계수 환경 설정하기

수동 계수를 수행하기 위해서는 '수동 계수자' 또는 '관리자' 권한을 가진 계정이 필요합니다. 따라서 사전에 '수동 계수자' 계정을 만들고, 동아리 부원들에게 배포해야 합니다. 이때 모든 사용자가 같은 ID/PW를 공유하지만, 수동 계수 기록을 중앙 서버에 전송할 때에는 이름을 지정하여 보내기 때문에 식별에는 어려움이 없습니다.

1. [계수기 서비스](https://counter.zetin.uos.ac.kr)에 로그인합니다.
1. '관리자 페이지 이동' > '사용자 관리' > '사용자 생성' 버튼을 누르고, 다음 내용을 참고하여 폼을 작성 후 '생성' 버튼을 누릅니다.
   - 이름: (예) 2025수동계수
   - 사용자명: ID에 해당하는 부분
   - 비밀번호: PW에 해당하는 부분
1. 생성된 사용자에 방패 모양 버튼을 눌러 권한을 수정하는데, '수동 계수자'에만 체크하여 저장합니다.
1. 대회 당일 수동 계수자 단체 톡방에 접속 방법을 공유합니다.
    ```
    https://counter.zetin.uos.ac.kr/

    1. 로그인(id: ______, pw: ______)
    2. 계수기 선택하기
    3. RPI Front Back IR 선택
    4. 수동 계수 버튼 클릭
    5. 기록자 이름에 본인 이름 기록
    6. 계수 시작 시 "시작", 정지 시 "정지", 리셋 시 "리셋" 버튼 누르기
    7. 수동 계수 정지하면 기록 전송 버튼 눌러서 전송하면 됨!
    ```
